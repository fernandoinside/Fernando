# Sobre o Portfólio

Este site reúne projetos de software prontos, documentados e disponíveis para licenciamento ou venda.

Cada produto possui uma página própria com detalhes sobre funcionalidades, tecnologias, documentação, demonstrações e condições comerciais.

Os repositórios de código podem permanecer privados. O objetivo desta página é apresentar o produto de forma clara para potenciais interessados.
