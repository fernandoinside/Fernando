# Portifólio de Produtos

Site Jekyll para apresentar produtos de software prontos, documentados e disponíveis para licenciamento ou venda.

## Estrutura

- `_projects/`: produtos cadastrados.
- `_layouts/`: layouts das páginas.
- `_includes/`: cabeçalho e rodapé.
- `pages/`: páginas institucionais.
- `docs/`: documentação de apoio.
- `examples/`: modelos para novos produtos.
- `assets/`: CSS, JS e imagens.

## Adicionar um novo produto

1. Copie `examples/modelo-produto.md`.
2. Cole em `_projects/nome-do-produto.md`.
3. Preencha os campos do front matter.
4. Escreva a documentação pública do produto no corpo do arquivo.
5. Teste localmente e publique no GitHub Pages.
